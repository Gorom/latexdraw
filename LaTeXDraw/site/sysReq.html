System Requirements



To launch LaTeXDraw you need, at least, jre 1.5_6 and 50 Mo of RAM. You can download the jre here: http://java.sun.com/j2se/1.5.0/download.jsp.
To create templates, you must the rights to write in the folder where LaTeXDraw is installed.
To execute a jar file, you can do a double click on it in your file explorer or by launching this command: java -jar latexdraw.jar

On Ubuntu and Debian, you need to install a package of fonts to be able to change the font of a text; you can get this package in command line with: apt-get install msttcorefonts
