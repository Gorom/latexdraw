Aidez le projet LaTeXDraw




Vous pouvez aider le Projet LaTeXDraw en :
  • Donnant votre opinion concernant LaTeXDraw ainsi que ses fonctionnalités PSTricks dans ce forum
  • Demandant l'implémentation de nouvelles fonctionnalités (PSTricks, interface, ...) dans ce forum
  • Nous signalant un bug dans le logiciel ou un problème dans le site (orthographe, ...) ici
  • Traduisant LaTeXDraw, ce site Web ou encore le manuel
  • Parlant (positivement) du Projet LaTeXDraw à votre entourage
  • Mettant un lien vers ce site depuis le votre
  • Faisant un don au Projet LaTeXDraw
