LaTeXDraw est un générateur de code PSTricks ou encore un éditeur de code PSTricks pour LaTeX distribué sous les termes de la licence GNU GPL. LaTeXDraw est développé en Java, il est donc indépendant du système d'exploitation. Il est disponible en Français, en Espagnol ainsi qu'en Anglais.
Vous devez avoir jre 1.6 au moins pour pouvoir utiliser LaTeXDraw.

latexdraw 2.0.3 disponible

Enfin ! latexdraw 2.0.3 vient juste de sortir et apporte une super (à mon avis) fonctionnalité : la possibilité d'exporter un dessin en pdf et en ps en utilisant votre distribution latex. En résumé, un document latex est créé à partir de votre dessin pour être ensuite compilé en un fichier pdf ou ps. Vous pouvez ensuite l'insérer dans votre document latex principal comme une image (les paquets pstricks ne sont ainsi plus nécessaire et la commande pdflatex peut être utilisée). jetez un oeil ici pour plus d'explications.
Cette version corrige également quelques problèmes génant (voir ici pour plus d'info). De plus, latexdraw est maintenant disponible en Italien !

Les principales fonctions de LaTeXDraw :

  • Plusieurs sortes de figures : rectangle, point, axes, losange, texte, dessin à main levé, ellipse, arc, grille, carré, camembert, courbe de Bézier, flèche, triangle, polygone, cercle, arc fermé, lignes jointes ;
  • Peut importer/insérer du code PSTricks ;
  • Peut insérer des images ;
  • Peut convertir des images dans plusieurs formats ;
  • Gestion des figures : rotation, redimensionnement, déplacement, positionnement ;
  • Gestion des couleurs ;
  • Multiselection ;
  • Plusieurs types de bordures : en pointillé, pleine ;
  • Plusieurs types de flèches ;
  • Plusieurs types de points ;
  • Gestion de l'épaisseur ;
  • Exportation en format eps, TeX, jpeg, ppm, bmp et png ;
  • Undo/redo ;
  • Gestion des zooms ;
  • Peut créer des modèles (des groupes de figures réutilisables) ;
  • Gestion des hachures ;
  • Grille magnétique
  • Miroir horizontal et vertical;
  • Gestion des doubles bords ;
  • Gestion de la position des bords des figures ;
  • Peut vérifier une nouvelle version de LaTeXDraw au démarrage ;
  • Peut être traduit ;
  • ...
