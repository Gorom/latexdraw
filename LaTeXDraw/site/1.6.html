LaTeXDraw 1.6 New and Noteworthy



Here are some of the more noteworthy things available in LaTeXDraw 1.6. See the release note for details about bugs fixed and other changes. Here the roadmap page of LaTeXDraw 1.6.



Compatibility



In general, the LaTeXDraw files are compatibles with 1.6, except in these case :
  • because of a bug in the version 1.5.1.1 and formers versions, the LaTeXDraw file containing grids cannot be opened with the 1.6 (an error will occured). See the release note for more informations ;

  • if a LaTeXDraw file contains figures with double borders and the language of the application is french, the double borders will be always 'inner'.


Import PSTricks File



LaTeXDraw can now import PSTricks file (latex files containing PSTricks commands). This first version of this PSTricks parser managed only few commands.
More informations about the PSTricks importation here.



Insert Pictures



LaTeXDraw can now insert pictures in a drawing.



Insert PSTricks figures



LaTeXDraw can now insert PSTricks figures in a drawing (using the PSTricks parser).



New Figure: joined Lines



A new kind of figure is managed by LaTeXDraw: joined lines are polygons without the last line.



Convert Pictures



LaTeXDraw can convert pictures from a format (jpg, png, gif, bmp) to an other format( eps, jpg, png, bmp, ppm).



More export



PNG, BMP, PPM formats are managed by LaTeXDraw to export drawing.
